Place URI
In the Syriac Gazetteer, a "place URI" is an HTTP Uniform Resource Identifier (URI), created by the Syriac Reference Portal project, that is used to identify uniquely a geographic place, irrespective of its location (which may be unknown or disputed), its names (which may not be unique), or any other attributes. Such URIs take the form http://syriaca.org/place/\d+ (where "\d+" indicates a string of one or more numerals). They are not to be confused with the HTTP URIs that identify individual pages or records in the Syriac Gazetteer itself.
Uniform Resource Identifier (URI)
"In computing, a uniform resource identifier (URI) is a string of characters used to identify a name or a web resource" (Wikipedia, s.v. "Uniform resource identifier"). The Syriac Reference Portal project uses URIs that conform to the standard HTTP URI Scheme (so-called HTTP URIs) to identify both web resources (like individual pages or records in online publications) and real-world resources like persons, places, and concepts (see e.g., Place URI).
